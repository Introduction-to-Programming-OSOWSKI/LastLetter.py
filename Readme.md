# **Last Letter**  

## **Assignment Description**  
Create a function called lastLetter() that returns the last letter of a given word w.

## **Examples**  
>lastLetter("minnesota")  
a  

>lastLetter("minnow")  
w  

## **Limitations**  
1 function  
0 for loops 
0 if statements  

## **Hints**
